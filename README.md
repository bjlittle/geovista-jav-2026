# geovista-jav-2026

Submission to the [Data Visualization and Effective Communication in Volcanology: Cross-disiplinary Lessons from Research and Practice](https://link.springer.com/collections/cgaadbacjg) collection in the [Journal of Applied Volcanology](https://link.springer.com/journal/13617).


## Clone the Repository

If you wish to explore either of the **Raikoke** and **Reykjanes** datasets, then first download our repository from GitHub:

```bash
> git clone https://github.com/bjlittle/geovista-jav-2026
> cd geovista-jav-2026
```

Now click either of the images below for further instructions.
